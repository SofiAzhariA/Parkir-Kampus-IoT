🛠️ Komponen Hardware

🧠
ESP32 DevKit V1
Mikrokontroler Utama
• CPU: Dual-core Xtensa LX6 240MHz
• RAM: 520KB SRAM
• WiFi: 802.11 b/g/n Built-in
• GPIO: 34 pin programmable
• Flash: 4MB
Mikrokontroler Utama
📡
HC-SR04 Ultrasonik
Sensor Jarak × 4
• Range: 2cm – 400cm
• Akurasi: ±3mm
• Frekuensi ultrasonik: 40kHz
• Pin: VCC, GND, TRIG, ECHO
• 1 unit per slot parkir
4 Unit (Slot A1,A2,B1,B2)
🖥️
LCD 16×2 I2C
Display Lokal
• 16 karakter × 2 baris
• Interface: I2C (SDA=GPIO21, SCL=GPIO22)
• Address: 0x27
• Backlight: LED biru
• Tampilkan status slot real-time
1 Unit
⚙️
Servo SG90
Palang Gate Otomatis
• Torsi: 1.8kg·cm
• Kecepatan: 0.1s/60°
• Control: PWM GPIO15
• Sudut buka: 90°, tutup: 0°
• Otomatis tutup saat penuh
1 Unit (Gate Masuk)
💡
LED Indikator
Indikator Slot + WiFi
• LED Merah × 4: Slot terisi
• LED Hijau × 1: Status WiFi
• Resistor 220Ω tiap LED
• Tegangan: 3.3V dari ESP32
• GPIO: 4, 5, 18, 19, 23
5 Unit
🔊
Buzzer Aktif
Sistem Alarm
• Tegangan: 3.3V – 5V
• GPIO: 2
• 1 beep: kendaraan masuk
• 2 beep: kendaraan keluar
• 3 beep: area penuh!
1 Unit

📋 Cara Import ke Wokwi

1
Buka Wokwi.com
Kunjungi wokwi.com dan login atau buat akun gratis
2
New Project → ESP32
Klik "+ New Project", pilih ESP32 sebagai board
3
Import diagram.json
Klik ikon "..." → "Import diagram" → upload file diagram.json
4
Paste sketch.ino
Salin isi file sketch.ino ke editor kode di tab kiri
5
Install Libraries
Library Manager → install: LiquidCrystal I2C, ESP32Servo, ArduinoJson
6
▶ Jalankan Simulasi
Klik tombol Play ▶ untuk mulai simulasi. Geser slider jarak di sensor untuk tes!

🔄 Logika Program ESP32

LOOP UTAMA (setiap 500ms)
1. Baca 4 Sensor HC-SR04
Ukur jarak → tentukan slot terisi/kosong
↓
2. Update LED Indikator
Merah=terisi, Mati=kosong
↓
3. Update LCD Display
Tampilkan "X/4 Terisi" pada layar
↓
4. Cek Kondisi Kritis
Jika PENUH → tutup gate + alarm buzzer
↓
5. Kirim Data ke Server (10s)
HTTP POST JSON → update dashboard web
FORMAT DATA JSON (HTTP POST)
{
  "timestamp": "2025-05-21T09:30:00",
  "device_id": "SMARTPARK-ESP32-001",
  "total_slot": 4,
  "slot_terisi": 3,
  "slot_tersedia": 1,
  "occupancy_rate": 0.75,
  "gate_status": "open",
  "slots": [
    { "id": "Slot A1", "terisi": true,  "jarak_cm": 10.5 },
    { "id": "Slot A2", "terisi": true,  "jarak_cm": 8.2  },
    { "id": "Slot B1", "terisi": true,  "jarak_cm": 12.1 },
    { "id": "Slot B2", "terisi": false, "jarak_cm": 180.0}
  ]
}
